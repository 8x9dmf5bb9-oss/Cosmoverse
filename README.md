# COSMOVERSE — Исследование Вселенной

![COSMOVERSE](https://img.shields.io/badge/COSMOVERSE-Исследование%20Вселенной-%2304030f?style=for-the-badge)

Современный, атмосферный одностраничный сайт космической тематики с богатой интерактивностью, анимациями и адаптивным дизайном.

[Открыть сайт](https://8x9dmf5bb9-oss.github.io/cosmoverse/)

## ✨ Особенности

- **Полностью интерактивный** космический интерфейс
- **Тёмная и светлая тема** с плавным переключением
- **Анимированный фон звёзд** (Canvas)
- **Параллакс-эффекты** в hero-секции
- **Анимированные счётчики** статистики
- **Слайдер миссий** с автопрокруткой
- **Прогресс-бары** с анимацией
- **Интерактивные вкладки** (Табы)
- **Аккордеон** в FAQ
- **Модальные окна** с информацией о команде
- **Валидация формы** обратной связи
- **Toast-уведомления**
- **Плавная навигация** и sticky-хедер
- **Полностью адаптивный** дизайн (мобильные устройства)

## 🚀 Демо

Откройте файл `index.html` в браузере — сайт работает без сборки и серверной части.

## 🛠 Технологии

- **HTML5**
- **CSS3** (с переменными, Grid, Flex, современными эффектами)
- **Vanilla JavaScript** (без фреймворков)
- Canvas API
- Google Fonts (Orbitron + Exo 2)
